Welcome to the ClaSS Installation script.

What you are doing.

You should have already unpacked class into the top-level of your web directory.

You should have already created a MySQL user account (with full privileges) to connect to the MySQL server.

You will need to give a name for the ClaSS database you want created and of a separate password for ClaSS to use when connecting to MySQL.

Need I say you should be running Firefox as your browser (with Javascript and Cookies switched to on).

In other words, you need to know what you are doing...
