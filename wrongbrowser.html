ClaSS

(ClaSS Student System)

ClaSS aims to be the ubiquitous classroom information assistant for schools. It is a complete student tracking, reporting and information management system.

ClaSS is designed for modern standards compliant browsers. If you don't have an up-to-date, xml-compliant browser then ClaSS will probably crash your browser, or at the very least ClaSS will neither look nor work correctly.

ClaSS is developed for, and works best with, Mozilla Firefox.
